# SWD4CS
 * SWD4CS stands for Simple WinForms Designer for CSharp (VSCode)
 * Visual Studio is heavy, so I often use VSCode, but it doesn't have a designer.
 * I had no idea how to create a designer, so I just wrote whatever came to mind.
 * This method won't work for complex things, but it might work for simple tools.
  
## Development environment
 * Windows11 Home  
 * VisualStudio2022 C# → VSCode .net10.0
